## 🆘 HELP & TROUBLESHOOTING

### If you're stuck on YOUR task:
1. Re-read your quick reference card
2. Check the detailed plan (EMERGENCY_2WEEK_SPRINT.md)
3. Ask in group chat
4. Ask the AI assistant (paste LLM_Context_Guide.md + your question)

### If your code isn't working:
1. Read the error message carefully
2. Check your imports
3. Test with simple mock data first
4. Ask Person 1 for help (integration issues)
5. Pair program with teammate

### If you're behind schedule:
1. Check PROGRESS_TRACKER.md - which tasks are incomplete?
2. Focus on CRITICAL tasks only
3. Skip NICE-TO-HAVE features
4. Ask team for help
5. Communicate honestly about status

### If the team is behind schedule:
1. Emergency team meeting
2. Assess what's actually critical
3. Cut scope if needed (see priority levels in main plan)
4. Redistribute work if someone is blocked
5. Increase daily communication
